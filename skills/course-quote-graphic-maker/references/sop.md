# 課程圖文製作 SOP

## 1. 目的與成品

把每節課的現場照片與逐字稿製作成一張兼具思想深度、紀錄真實性與系列一致性的公開圖文。預設每節課一張，輸出 16:9、2048 × 1152、高解析 PNG。

## 2. 輸入

基本資料：

1. 一張或多張課程照片。
2. 該堂課逐字稿。
3. 課程實際日期。
4. 品牌或系列名稱；若未提供，使用 `課程精選`。

如果日期已存在於訊息、前後文、檔名、課程表或可靠 metadata，直接使用。只有日期缺失、矛盾或會影響成品正確性時才詢問。堂次、時段、人物角色與姓名不屬於圖面必備資料；即使已知，也不要顯示。

## 3. 逐字稿處理

### 理解

先完整閱讀，判斷核心主題、最值得記住的觀點、能引導反思的句子，以及單純口語轉場。

### 允許

- 修正明顯語音辨識錯字。
- 移除口語贅詞與重複。
- 整理標點、語序及斷句。
- 在不改變原意下精簡。

### 禁止

- 增加講者沒有表達的觀點。
- 為了文學感改變原意。
- 將不確定內容自行補成事實。
- 把整段逐字稿縮小塞入畫面。

### 文字層級

1. 頁首：只有使用者指定的品牌或系列名稱；未提供時使用 `課程精選`。
2. 核心主句：約 12–30 字，兩至三行。
3. 延伸理解：約 30–70 字，三至五行；可含一至兩個思辨問題。
4. 左下角：日期，沒有其他短句。

若逐字稿沒有足以支持的問題，不強行創造問題。若辨識錯誤會改變意思，停止猜測並詢問。

## 4. 日期

顯示格式固定為 `YYYY.MM.DD`。

判定順序：

1. 使用者提供的課程日期。
2. 課程表或活動資訊。
3. 照片原始拍攝日期。
4. 詢問使用者。

日期放在左下角原短句位置，上方可保留細分隔線。不得加「西元」、星期、日期前綴、結語或第二段金句；不得以製作日冒充課程日。

## 5. 照片

### 原始檔

- 永遠保留，不覆寫。
- 從工作副本開始。
- 多張照片時，選擇表情自然、對焦準確、主體清楚且有文字空間者。

### 允許調整

- 裁切、縮放與重新定位整張照片。
- 亮度、對比、色溫、飽和度及局部明暗。
- 細微暗角與質感顆粒。

### 真實性邊界

不得改變人物五官、年齡、髮型、表情、姿勢、手指、服裝、名牌、麥克風、投影或牆面文字。生成式工具只可探索，不得以改變紀錄內容的版本作為成品。所有中文字另行排版。

## 6. 構圖與視覺

固定視覺：

- 左文右人。
- 滿版原始照片。
- 左側深墨綠或深藍綠漸層文字區。
- 米白主文、淡金重點、低飽和薄荷綠問題句。
- 主句用宋體或明體；品牌字樣及日期用黑體。
- 整體沉靜、精緻、留白、有文化感，不做商業促銷式效果。

人物安全規則：

- 講者位於右側明亮區。
- 蒙板在碰到人物前完全淡出。
- 臉、頭髮、上半身、手與麥克風沒有可察覺的暗色覆蓋。
- 右側保留呼吸空間。
- 優先重新裁切與定位整張照片，不切出人物搬移。

若照片無法支援左文右人且人物安全，要求替代照片，不鏡像現場、不生成新背景、不勉強遮人。

## 7. 排版順序

1. 建立照片工作副本。
2. 確認人物、清晰度、表情與文字安全區。
3. 整理並核對主句與延伸內容。
4. 將人物定位於右側。
5. 建立左側漸層，確認避開人物。
6. 在左上角放入品牌或系列名稱，不加入任何其他資訊。
7. 放入主句與延伸內容。
8. 在左下角放入日期。
9. 調整斷行、字級、行距與留白。
10. 輸出 PNG，實際開啟檢查。

## 8. QA

### 內容

- 左上角逐字顯示使用者指定的品牌或系列名稱。
- 全圖沒有堂次、時段、人物角色或分享者姓名。
- 精選文字忠於逐字稿。
- 繁體中文、標點及日期正確。
- 左下角只有 `YYYY.MM.DD`。

### 視覺

- 縮圖第一眼看見主句，第二眼看見講者。
- 人物明亮，沒有蒙板薄膜。
- 文字不遮人、不裁切、不擁擠。
- 日期清楚但不搶主視覺。
- 膚色自然、現場感真實。

### 技術

- 2048 × 1152 PNG。
- 字型無遺失，沒有亂碼。
- 檔案可開啟。
- 原始照片未被覆寫。
- 必須查看實際輸出，不能只確認檔案存在。

任何一項失敗都先修正，再交付。

## 9. 版本與交付

檔名：

`YYYY-MM-DD_照片檔名_課程精選圖文_v01.png`

修訂使用 `v02`、`v03`。確認後使用：

`YYYY-MM-DD_照片檔名_課程精選圖文_定稿_v03.png`

不覆蓋原始照片與已確認版本；工作 HTML、設定 JSON、探索稿和正式 PNG 分開存放。

每次交付：

1. 顯示實際預覽。
2. 提供高解析 PNG 連結。
3. 列出核心主句與課程日期。
4. 若素材有限制，簡短說明。

## 10. 停止條件

遇到以下情況只詢問完成任務所需的一至兩個問題：

- 無法確認課程日期。
- 逐字稿破碎到可能誤解原意。
- 照片解析度不足或無法兼顧文字與人物。
- 必須生成新背景或改變人物才能完成。
- 使用者要求改變整個系列固定視覺。
